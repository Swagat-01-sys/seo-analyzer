# 📊 SEO Scoring Methodology

## Overview

The SEO Analyzer evaluates websites using four primary categories. Each category contributes a weighted percentage toward the final SEO Score (0–100).

---

# Score Distribution

| Category          | Weight |
| ----------------- | ------ |
| Technical SEO     | 30%    |
| Metadata Analysis | 20%    |
| Content Analysis  | 30%    |
| Performance       | 20%    |

---

# Technical SEO

The Technical SEO module checks whether a website follows fundamental search engine optimization best practices.

Checks include:

* HTTPS
* Robots.txt
* Sitemap.xml
* Canonical URL
* Structured Data
* Indexability
* Security Headers

Each successful validation increases the technical score.

---

# Metadata Analysis

Metadata helps search engines understand page content.

The analyzer evaluates:

* Title Tag
* Meta Description
* Meta Keywords
* Open Graph Tags
* Twitter Cards

Missing metadata reduces the category score.

---

# Content Analysis

Content quality is measured using:

* Word Count
* Character Count
* Readability
* Keyword Frequency
* Top Keywords

Well-structured and informative content contributes to a higher SEO score.

---

# Performance Analysis

Performance metrics include:

* Response Time
* Page Size
* Compression
* Server Information

When Google Lighthouse executes successfully, additional metrics are collected:

* Performance Score
* Accessibility
* Best Practices
* SEO Score
* First Contentful Paint (FCP)
* Largest Contentful Paint (LCP)
* Speed Index
* Total Blocking Time (TBT)
* Cumulative Layout Shift (CLS)

---

# Recommendation Engine

Recommendations are generated whenever important SEO optimizations are missing.

Examples include:

* Add Meta Description
* Create Sitemap.xml
* Implement Structured Data
* Improve Page Speed
* Add Image ALT Attributes

---

# Final Score

The weighted category scores are combined to produce an Overall SEO Score between **0 and 100**.

Higher scores indicate stronger adherence to modern SEO best practices and improved search engine visibility.
